# claude-skills リポジトリ

foo-543674 の設計思想・コードレビュー観点・実装方針を Claude Skills として定義・管理するリポジトリ。

## リポジトリ構成

```
claude-skills/
├── README.md       # 人間向け（セットアップ手順など）
├── CLAUDE.md       # このファイル
└── skills/         # スキル本体
    └── <skill-name>/
        ├── SKILL.md        # エントリポイント（YAML frontmatter + 指示）
        └── resources/      # 詳細なリファレンス、チェックリスト等
```

## スキルの作成・編集ルール

### SKILL.md の形式

```yaml
---
name: skill-name
description: このスキルが何をするか、いつ使うか
---

# スキル名

指示内容をここに記述する。
```

- `name`: スキル名
- `description`: Claude が自動で関連性を判定するために使う説明文

### 守ること

- SKILL.md は 500 行以内に収める。詳細は `resources/` に分離する
- 1 スキル 1 フォルダ。目的が異なるものは別スキルにする
- スキルを追加・変更したら README.md のスキル一覧も更新する

## フィードバックの反映

foo-543674 からスキルの書き方や内容について指摘・修正を受けた場合、同じ指摘が繰り返されないように以下を必ず行う:

1. **対象スキルの修正**: 指摘内容をスキルに反映する
2. **CLAUDE.md の更新**: 指摘が他のスキルにも適用できる一般的なルールであれば、このファイルの「守ること」や該当セクションにルールとして追記する
3. **既存スキルへの横展開**: 追加したルールに違反している既存スキルがあれば、それも合わせて修正する
